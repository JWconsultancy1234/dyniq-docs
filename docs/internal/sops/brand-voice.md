# Brand Voice Guide

All DYNIQ customer-facing content follows the **Everyman + Caregiver** archetype. Approved by CEO Walker via board meeting (2026-02-10, 8 agents, weighted vote 3.5).

## Brand Identity

| Element | Value |
|---------|-------|
| **Archetype** | 60% Everyman + 40% Caregiver |
| **ICP** | "The Busy Installer" (35-55, HVAC/plumbing/electrical, Flanders) |
| **Ruben** | "Uw Digitale Collega" (Your Digital Colleague) |
| **Promise** | "10 leads in 30 dagen of geld terug" (with conditions) |
| **Pricing** | EUR 997/mo (shown publicly) + EUR 831/mo annual (2 months free) |
| **Colors** | Emerald #10B981 (primary) + Deep Emerald #059669 (CTA gradient) |
| **Theme** | 70/30 dark/light hybrid |

## The Everyman Voice (60%)

Speaks like a trusted colleague, not a salesperson. Practical, proven, simple.

| Do | Don't |
|----|-------|
| "Werkt" (it works) | "Transformeer" (transform) |
| "Simpel" (simple) | "Magie" (magic) |
| "Helder" (clear) | "Visie" (vision) |
| "Betrouwbaar" (reliable) | "Droom" (dream) |
| "Bewezen" (proven) | "Toekomst" (future) |
| "Meer klanten" (more customers) | "Unlock verborgen klanten" |

## The Caregiver Voice (40%)

Shows genuine care for the installer's success. Protective, supportive.

| Do | Don't |
|----|-------|
| "Wij zorgen voor..." (we take care of) | "U moet..." (you must) |
| "Gegarandeerd" (guaranteed) | "Misschien" (maybe) |
| "Geen gedoe" (no hassle) | Technical jargon |
| "Geld terug" (money back) | Fine print hiding |

## Ruben Positioning

| Element | Guideline |
|---------|-----------|
| **Title** | "Uw Digitale Collega" (ALWAYS) |
| **Never say** | "AI Employee", "Digital Sales Expert", "Robot", "Bot" |
| **Personality** | Friendly colleague, reliable, punctual |
| **Tone** | Casual but respectful (Belgian Dutch) |

## Color Usage

| Color | Hex | When to Use |
|-------|-----|-------------|
| Emerald | #10B981 | CTAs, links, highlights, primary actions |
| Deep Emerald | #059669 | CTA gradients, pressed states |
| Dark | #0F1115 | Hero, comparison, guarantee, CTA sections (70%) |
| Light | #F9FAFB | HowItWorks, BeforeAfter, Testimonials (30%) |

:::warning Deprecated Colors
Never use Cyan #00D4FF or Purple #8B5CF6 (dropped per board meeting 2026-02-10).
:::

## CTA Guidelines

| CTA Type | Text | Target |
|----------|------|--------|
| Primary | "Bekijk live demo" | #demo section |
| Secondary | "Bel nu" | Phone number |
| Tertiary | "Boek strategie-sessie" | Cal.com link |
| ROI | "Bereken uw ROI" | #roi-calculator |

**Never use:** "Start gratis trial", "Sign up", "Get started" (too SaaS-like for ICP)

## Guarantee Conditions

The guarantee MUST include conditions (per CFO analysis, 10-15% refund risk):

1. Volledige terugbetaling (full refund)
2. Schriftelijke garantie (written guarantee)
3. Voorwaarden: 24u reactietijd op leads, onboarding afgerond

**Never say:** "Geen vragen gesteld" (no questions asked)

## Pre-Publish Checklist

- [ ] Uses "we" not "I" (team positioning)
- [ ] No magic/transformation language
- [ ] Ruben called "Uw Digitale Collega"
- [ ] Colors use emerald + dark (no cyan/purple)
- [ ] Practical, proven language (Everyman voice)
- [ ] Shows care for installer's success (Caregiver voice)
- [ ] Pricing visible if relevant (EUR 997/mo)
- [ ] Guarantee includes conditions
- [ ] Belgian/Flemish identity present
- [ ] Truth Audit completed (all claims verified)
